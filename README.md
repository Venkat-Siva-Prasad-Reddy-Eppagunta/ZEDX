## Project Overview

ZEDX is a fintech-focused productivity mobile application designed to help users stay focused and intentional while working toward financial and task-oriented goals.

This repository contains the frontend of the ZEDX application, built using Expo and React Native. The frontend is fully API-driven and communicates with the ZEDX backend via REST APIs.


## Key Features

- Secure authentication flow
- Clean and distraction-free mobile UI
- File-based routing using Expo Router
- Modular and reusable components
- Environment-based configuration
- API-driven frontend architecture

## Installation

Prerequisites:
- Node.js (v18+)
- npm or yarn
- Expo CLI

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Venkat-Siva-Prasad-Reddy-Eppagunta/ZEDX.git
2. Navigate to the project directory:
   ```bash
   cd zedx
3. Install dependencies using Composer:
   ```bash
   npm install
4. Create a .env file:
   ```sh
   EXPO_PUBLIC_API_URL=http://localhost:5000
5. Running the App
   ```sh
   npx start

Run using:
- Expo Go
- Android Emulator
- iOS Simulator
- Web browser


## Usage

- Users authenticate using backend APIs
